# Timelapse Microscope
Automated timelapse capture system with live camera streaming and a kiosk display interface.

A very simple, quickly‑built timelapse and streaming setup designed for a Raspberry Pi (or any Linux system). It was created to capture bacterial growth samples so children could check on their experiment remotely after the samples were collected and stored in a controlled environment.

This is not a polished or production‑grade project — just a practical solution that gets the job done.

#Captures an image every 60 seconds

- Compiles a daily timelapse video.
- Provides optional live streaming using MJPG‑Streamer
- Includes a basic kiosk‑mode webpage to show the live image/stream
- Runs on lightweight hardware with minimal setup

Why This Exists
This was built quickly to support a school experiment where students needed to:
- Leave bacterial samples in a controlled environment
- Check progress remotely
- Watch a timelapse of growth over the day
- Optionally view a live stream

It’s intentionally simple — just enough to work reliably for a short project. The code is basic and functional, not polished

Improvements are welcome if anyone wants to build on it
